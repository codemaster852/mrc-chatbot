# MRC Chatbot

This is a simple AI chatbot using the Forefront API with a React frontend and a Node.js backend.

## Installation

### 1. Backend Setup
```sh
cd backend
npm install
node server.js
```

### 2. Frontend Setup
```sh
cd frontend
npm install
npm start
```

## Deployment
- Deploy the frontend on **Vercel/Netlify**.
- Deploy the backend on **Render/Railway/Vercel**.
